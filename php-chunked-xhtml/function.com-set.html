com_set

(PHP 4)

com_set — 给 COM 组件的属性赋值

说明

Deprecated, use the OO syntax instead.

Example #1 不要使用 com_set() ，使用 OO 语法代替

<?php
   // do this
   $obj->property = $value;
   // instead of this:
   com_set($obj, 'property', $value);
   ?>

注释

Note: 此函数在 PHP 5 中不再存在. 你应该使用更自然的面向对象语法操作属性及方法。
